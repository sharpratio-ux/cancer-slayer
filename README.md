🧬 나나와 암세포 죽이기 (Cancer Slayer: Nano Warfare)
차세대 CAR T 세포 및 ADC 정밀 타격 시스템을 이용한 암세포 박멸 시뮬레이션

🎮 작전 개요
박사님(사용자)은 최첨단 바이오 기술인 CAR T 세포(Chimeric Antigen Receptor T-cell) 부대의 지휘관이 되어, 주인공 '나나'와 함께 인체 내 악성 종양을 제거해야 합니다. 18개 스테이지에 걸쳐 갑상선, 대장, 신장, 간, 위, 폐에 침투한 암세포를 CAR T 세포와 ADC(Antibody-Drug Conjugate) 미사일로 정밀 타격하여 환자의 생명을 구원하십시오.

🚀 주요 기능 (Key Features)
차세대 면역 항암 기술: 유전자를 개량한 CAR T 세포를 배양 및 투입하여 암세포를 직접 타격합니다.

ADC 정밀 미사일 시스템: 항체-약물 접합체(ADC) 기술을 활용한 강력한 광역 공격 시스템을 가동합니다.

실시간 바이오 모니터링: 9:16 세로형 인터페이스를 통해 환자의 장기별 종양 증식 상태를 실시간으로 지휘합니다.

바이오 강화 로드맵: 작전 포인트(P)를 사용하여 CAR T 세포의 공격력(ATK), 배양 속도(SPD), 생존력(HP)을 최적화합니다.

임상 완료 피날레: 6개 장기, 18스테이지의 모든 타겟을 완치했을 때 나타나는 특별한 엔딩 시퀀스를 확인하십시오.

🛠 기술 스택 (Tech Stack)
Core: Pure JavaScript (Canvas API 기반 정밀 연산)

Platform: PWA (Service Worker를 통한 오프라인 작전 지원)

Deployment: GitHub Pages (HTTPS 보안 호스팅)

📂 실행 방법
접속 주소: https://[박사님ID].github.io/cancer-slayer/

브라우저 메뉴에서 '홈 화면에 추가' 또는 **'앱 설치'**를 눌러 모바일 기기에 바로 설치하십시오.

아이콘을 클릭해 작전 오퍼레이터 나나의 브리핑에 따라 임무를 시작합니다.
